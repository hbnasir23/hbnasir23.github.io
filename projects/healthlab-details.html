Project Overview

HealthLab is a comprehensive health monitoring application designed to integrate with wearable bands for real-time vital tracking. The app provides role-based dashboards for patients, doctors, and administrators, offering a complete healthcare ecosystem in one platform.

Key Features

  • Real-time vitals monitoring from wearable devices
  • Role-based dashboards (Patient, Doctor, Admin)
  • Telehealth functionality with video consultations
  • Appointment booking system with calendar integration
  • Pharmacy module for medication tracking
  • Health data visualization with charts and trends
  • Emergency alert system for critical health events

Technologies Used

Frontend

  • Flutter Framework
  • Dart Programming Language
  • Provider State Management
  • Custom UI Components

Backend

  • Supabase Authentication
  • PostgreSQL Database
  • Supabase Realtime API
  • RESTful API Integration

Other Technologies

  • Firebase Cloud Messaging
  • WebRTC for Video Calls
  • Chart.js for Data Visualization
  • Google Maps API

Challenges & Solutions

Real-time Data Synchronization

Ensuring seamless real-time updates across all devices was challenging. Implemented Supabase's realtime subscriptions to listen for database changes and update the UI accordingly.

Wearable Device Integration

Different wearable brands had varying communication protocols. Created a modular architecture with adapters for each device type, with Bluetooth Low Energy as the common interface.

Role-based Access Control

Managing different user roles with appropriate permissions required careful planning. Implemented row-level security in PostgreSQL and role checks at both client and server levels.
